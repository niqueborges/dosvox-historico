# Repositório Histórico do DOSVOX (Fossil & Original State)

**Propósito:**
Este repositório é destinado **exclusivamente** à preservação do arranjo físico original do DOSVOX, assim como foi encontrado na data da extração primária. 

Ele **não** representa uma arquitetura idealizada, refatorada ou reconstruída. O código aqui dentro não foi limpo, nem organizado por responsabilidade. 

**Objetivo Científico:**
Seu objetivo é preservar evidências históricas. A manutenção desse "sítio arqueológico" intacto permite que futuros pesquisadores e desenvolvedores possam verificar as inferências documentadas ou encontrar novos padrões sem correrem o risco de perder artefatos que foram apagados por refatorações modernas ou extrações taxonômicas.
