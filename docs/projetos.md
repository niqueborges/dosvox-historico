# Módulos e Aplicativos do Sistema DOSVOX

Baseado no inventário da pasta de instalação (`C:\winvox`), mapeamos as dezenas de ferramentas independentes que compõem o ecossistema DOSVOX. O sistema engloba desde editores de texto básicos até jogos e comunicação em rede.

## Núcleo e Sistema Base
- **dosvox.exe**: O orquestrador principal, porta de entrada e menu do sistema.
- **config.exe / ajustaIni.exe**: Gestão de configurações e preferências do usuário.
- **atuvox.exe**: Atualizador do sistema.
- **ligavox.exe / desliga.exe**: Aplicativos para controle e inicialização/desligamento seguro do ambiente e da fala.

## Produtividade e Edição
- **edivox.exe**: O clássico e poderoso editor de texto voltado para interface falada.
- **letrix.exe**: Programa de aprendizado e digitação.
- **txtword.exe**: Integração/conversão para documentos padrão Word.
- **agenda.exe / agenvox.exe**: Calendário e compromissos.
- **calcuvox.exe**: Calculadora falante.
- **planivox.exe**: Planilha eletrônica acessível.
- **fichavox.exe**: Gerenciador de fichários/banco de dados simples.
- **epubvox.exe**: Leitor de livros no formato e-Pub.

## Internet e Comunicação
- **webvox.exe / miniweb.exe**: O navegador web acessível que converte páginas em estruturas lineares lidas em voz alta.
- **cartavox.exe / cartex.exe**: Clientes de e-mail com gerenciamento em pastas (provável uso de `imaputil.exe`).
- **papovox.exe**: Bate-papo exclusivo para a rede de usuários DOSVOX.
- **mircvox.exe / tnetvox.exe**: Clientes para IRC e Telnet.
- **ftpvox.exe**: Cliente para transferência de arquivos via FTP.
- **voxnews.exe**: Leitor de notícias/RSS.
- **googlevox.exe**: Interface limpa e acessada para buscas no Google.

## Multimídia e Ferramentas
- **midiavox.exe / cdmp3.exe / radio50.exe**: Players de áudio, CD e rádios online (usando motores `.dll` e FFmpeg).
- **voxtube.exe**: Navegador especializado em áudio/vídeos do YouTube.
- **dvtxt.exe / pdftotext.exe**: Ferramentas focadas na extração de texto de PDF e imagens (via OCR) para torná-los legíveis.
- **traduvox.exe / ttrad.exe / dicio.exe**: Tradutores e dicionários offline/online (com suporte das bases `.dic` e `.exc`).

## Jogos (Lúdicos e Didáticos)
O DOSVOX possui uma vasta biblioteca de jogos e passatempos em formato totalmente de texto/áudio espacial:
- **jogavox.exe**: Hub central de jogos.
- **colossal.exe**: Jogo de aventura em texto.
- **chessvox.exe**: Xadrez falado.
- **sudovox.exe**: Sudoku adaptado.
- **forcavox.exe / cruzavox.exe / palavrox.exe**: Jogos de adivinhação, palavras cruzadas e forca.
- **memovox.exe / memojogo.exe**: Jogos da memória sonoros.
- **casino.exe / suecavox.exe / baralhovox.exe**: Jogos de cartas e azar.
- **lunarvox.exe / piratvox.exe / nimvox.exe**: Mini-jogos clássicos portados para a plataforma.

## Síntese e Braille
- **braivox.exe / brlServer.exe**: Serviços para integração com hardwares de displays/linhas Braille e formatação Braille.
- **sapiutil.exe**: Módulo para vozes externas SAPI do Windows.
