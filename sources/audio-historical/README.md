# Áudios Históricos (Historical Audio)

Este diretório contém uma amostra restrita e representativa dos áudios estruturais do DOSVOX. Milhares de arquivos .wav educacionais e jogos foram deliberadamente ignorados no repositório Git, sendo mantidos apenas nos instaladores originais (fora de controle de versão).

Aqui preservamos apenas:
1. Sons clássicos do sistema (ex: bips de navegação).
2. Mensagens gravadas estruturais originais.
3. Áudios de treinamento historicamente vitais.
