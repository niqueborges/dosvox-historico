# Binários Órfãos (Orphan Binaries)

Este diretório preserva binários clássicos (.exe, .dll) para os quais o **código-fonte foi perdido no tempo** ou nunca existiu (softwares de terceiros da era MS-DOS), mas que são vitais para a historicidade e funcionamento da ponte arquitetural do DOSVOX.

Ao adicionar um binário aqui, edite este arquivo e documente:
1. Origem
2. Função
3. Ausência do código-fonte (confirmada)
4. Motivo da preservação
