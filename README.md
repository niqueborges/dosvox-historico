# DOSVOX Histórico

Projeto independente de documentação, preservação e pesquisa sobre o ecossistema DOSVOX, desenvolvido pela UFRJ desde 1994.

## Objetivos

- Preservar informações históricas do projeto.
- Catalogar componentes e bibliotecas.
- Registrar autores e colaboradores.
- Mapear a arquitetura do sistema.
- Recuperar documentação e links antigos.
- Facilitar pesquisas futuras sobre acessibilidade e computação assistiva no Brasil.

## Fontes principais

- Portal DOSVOX
- Código-fonte distribuído pela UFRJ
- Publicações acadêmicas
- Entrevistas e vídeos
- Documentação recuperada
