# DOSVOX Shell

**Propósito:**
Destinado ao menu principal e aos utilitários que implementam a experiência nativa de "sistema operacional virtual" do DOSVOX. Nenhum código foi extraído fisicamente para este diretório ainda.

**Critério de Inclusão:**
O Shell abrange os programas que fornecem a navegação de arquivos, testes de sistema, e as rotinas que dão a sensação de se estar operando o computador (como as listagens de discos e diretórios lidas pelo DOSVOX).

**Exemplos mapeados na taxonomia:**
- `dosvox.dpr` (Menu Principal)
- Utilitários de sistema (`ajustaIni`, gerenciadores de arquivos integrados).
